# Form Validator

A simple and responsive **Form Validation** project built using **HTML, CSS, and JavaScript**. This project validates user input on the client side and displays appropriate error messages before form submission.

## 🚀 Live Demo

https://bandarusasidhar.github.io/form-validator/

> *(Update the link if your GitHub Pages URL is different.)*

---

## 📌 Features

- ✅ Required field validation
- ✅ Username validation
- ✅ Email format validation
- ✅ Password validation
- ✅ Confirm password validation
- ✅ Instant error messages
- ✅ Success validation feedback
- ✅ Clean and responsive UI
- ✅ Pure JavaScript (No libraries)

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Bandarusasidhar/form-validator.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

No installation is required.

---

## 📖 Validation Rules

- Username cannot be empty.
- Email must be in a valid format.
- Password must meet the required criteria.
- Confirm Password must match the Password.
- Errors are displayed below the respective input fields.
